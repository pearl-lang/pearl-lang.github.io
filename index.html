Pearl language

A simple, elegant, and powerful programming language.

About Pearl

Pearl is designed to be easy to learn and use, with a focus on readability and simplicity. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming. It's simple to build eBPF programs with Pearl.
